# Rock Paper Scissors Game
This is a simple command-line interface (CLI) game where a player can play Rock, Paper, Scissors against the computer.
## How to Play
1. Run the `main.py` script.
2. When prompted, enter your choice: Rock, Paper, or Scissors.
3. The computer will randomly select its choice.
4. The game will determine the winner based on the classic rules:
   - Rock crushes Scissors
   - Paper covers Rock
   - Scissors cuts Paper
5. If both players choose the same option, the game will prompt you to choose again until there is a winner.
